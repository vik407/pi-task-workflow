# Lessons

Capture durable corrections from the developer here.

When the developer corrects the assistant, append:

```markdown
### YYYY-MM-DD - Brief title

- **Trigger**: What happened.
- **Correction**: What the developer directed.
- **Rule**: Prevent this by always doing X.
- **Area**: Prompt, skill, script, task workflow, documentation, or implementation.
```
